# PetMatchAI - How to Use

## First Launch Experience

### 1. Onboarding Screen
- Welcome screen with "Get Started" button
- Swipe through 3 introduction slides about AI matching

### 2. Lifestyle Quiz (10 Questions)
1. Home type: Apartment / House with Yard / House without Yard
2. Do you have a yard? Yes/No
3. Do you have children? Yes/No
4. Do you have other pets? Yes/No
5. Do you work from home? Yes, full time / Sometimes / No
6. Hours pet will be alone: Less than 4 / 4-6 hours / More than 6
7. Monthly pet budget: Under $50 / $50-$100 / $100-$200 / $200+
8. First-time pet owner? Yes/No
9. Preferred size: Small / Medium / Large / No preference
10. Energy level: Low / Moderate / High / No preference

### 3. Home Screen
- Shows "Your Top Matches" with pet cards
- Match percentage displayed on each card
- Pull to refresh for new matches

### 4. Browse Screen
- Grid view of all available pets
- Filter by species: Dogs, Cats, Birds, Small Animals
- Search by breed or name
- Sort by: Match %, Distance, Age

### 5. Pet Detail Screen
- Hero photo with swipe gallery
- Pet info: Name, breed, age, size, gender
- Compatibility score with explanation
- Personality traits icons
- Health & care info
- Shelter information
- "Add to Favorites" button
- "Contact Shelter" button

### 6. Profile Screen
- User preferences summary
- Edit quiz answers
- Theme toggle (Light/Dark)
- Subscription status

### 7. Settings Screen
- Notifications toggle
- Location services
- Theme selection
- Privacy policy
- Terms of service
- Export data
- Upgrade to Premium
- Delete account

---

## Testing Flow

1. **Complete Quiz** → Answer all 10 questions
2. **View Matches** → See AI-generated match percentages
3. **Browse Pets** → Explore all available pets
4. **View Details** → Check pet profile, compatibility score
5. **Add to Favorites** → Save a pet to your list
6. **Toggle Theme** → Switch between light/dark mode
7. **Upgrade Premium** → Test subscription flow
